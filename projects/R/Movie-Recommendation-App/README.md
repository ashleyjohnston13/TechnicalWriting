# Movie Recommendation App (R Shiny)

This project is an interactive movie recommendation tool built in R using the Shiny framework. The app allows users to select movies they enjoy and receive personalized recommendations generated from association rules.

The goal of this project was to create a simple, intuitive interface that demonstrates how data-driven recommendations can be made using real-world methods like market basket analysis.

## ðŸ§  How It Works

1. The user selects movies they enjoy.  
2. The app filters association rules to find strong co-occurrences.  
3. Recommendations are sorted by confidence or lift.  
4. Results are displayed as a clean, simple output table.  

The app uses reactive expressions to update results instantly based on user selections.
